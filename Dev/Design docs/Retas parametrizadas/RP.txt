PARÂMETROS:

(x', y') = f(x, y, A, t)

bulletWord 2:

t = 4 bits (0-15) ; parametrização da reta
A = 7 bits (0-79) ; ângulo da reta

MECANISMO:

TABELAdx é uma matriz de 80 x 16 valores, um para cada ângulo e parâmetro. A fórmula para acessar um valor é A*16 + t. 

TABELAdy é uma matriz de 80 x 16 valores, um para cada ângulo e parâmetro. A fórmula para acessar um valor é A*16 + t

TABELAtmax é um array de 80 valores, um para cada ângulo e parâmetro. A fórmula para acessar um valor é A

ALGORITMO:

Load bulletWord 2
Get A
Get t
Offset = A*16 + t

Get x
dx = load (TABELAdx + Offset)
x' = x + dx
Set x'

Get y
dy = load (TABELAdy + Offset)
y' = y + dy
Set y'

t += 1
tmax = load (TABELAtmax + A)
if(t == tmax) t = 0
Set t

@SET r0, r4, EB2RP.y ; set y'

loadi r3, r3 ; tmax

jeq skip_zerar_t_reta_parametrizada
